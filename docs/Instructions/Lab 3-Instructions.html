Lab #3

 

 

Create a Web page that contains FOUR hyperlinks using images. One link should direct the User to an existing page in your Web site ( ie. to the page for Lab 1 ), and the other three image links should take the user to an "outside" Web site's Home Page ( your choice ).

 

Now, add a TEXT LINK next to each of your four image links.  These links should also access the SAME three sites you have chosen to link to with your images. You are doing this so that your web page visitor will have another way to access your linked pages should some problem arise and your images fail to “launch.” These text links should also be used to inform your visitor what the links do/their destination (which sites they link to). For example, “Click here to visit my first web page,” or “Click here to visit eBay!”

TIP:  The pictures should reflect the subject that they are linking to and tie into the theme of your Web site! For this same reason, you might also want think about being consistent in the use of colors and fonts as you build each page.

 

Also, be sure to add a heading at the top of the page such as “Lab 3,” “Image Link Page,” Image Links,” etc. This needs to be treated as a professional-looking page.
