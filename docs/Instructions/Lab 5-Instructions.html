Lab # 5

 

Create a web page with STYLE!!!!

Your completed web page will contain both EMBEDDED and INLINE styles.

 

You will need to use the various style properties to first create an EMBEDDED style for each of the following tags: P, H1, H2, H3, H4, H5, H6, SPAN and DIV.  EACH style MUST consist of at least 4 different properties. You are not to use the same 4 properties for all of the listed tags. You may repeat a style property but not more than once!

 

Next, you will use each of the NINE tags on your Web page to display the styles that you have created. To do this, you just need to simply use the tag along with some text to display your style. What the content consists of is up to you!

 

Then, after using each of the tags once to display your EMBEDDED styles, you will use each of them a SECOND time.

 

This time, however, you will use the INLINE style structure to change ONE of the style property’s VALUE that you had just assigned in your  EMBEDDED style.

 

So, for example, if you had assigned the <p> tag this style:  font-family: arial , you might change it’s VALUE to: font-family: impact. You should have assigned each of the NINE tags listed above at least FOUR style properties, so you only need to take ONE of those 4 properties and assign it a new VALUE in the INLINE style format. Obviously, this will require you to use the SAME NINE TAGS a second time on your page with the INLINE style code added.

Again, the actual text you use is up to you. You will be graded on whether or not the styles assigned to each of your selectors (tags) display correctly and whether or not you have included all of the various style attributes as described above.
