The image must be LARGE!

 

This image needs to be added to a well-designed Web page containing a background, title, links, corresponding text, etc. You MUST make it look PROFESSIONAL!

 

You must create at least FIVE hotspots and each of the three basic shapes MUST be used at least once ( ie. circle, polygon and rectangle). The destination must be DIFFERENT for each HOTSPOT created.
