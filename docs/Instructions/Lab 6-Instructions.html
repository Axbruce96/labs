Lab # 6

 

Create THREE web pages PLUS ONE style sheet that works with CSS.

First, you need to create an individual style sheet. On it, you will define styles for ( minimum ) FIVE – SEVEN different selectors ( HTML tags ). You will assign each one different properties so that you use FOUR – SIX different properties per selector/tag. You need to use a variety of properties and NOT the same ones used repeatedly (and yes, they may be used in any combination)!  After you have created this style sheet, you will need to LINK it into two of your web pages and IMPORT it into the third.


ALL THREE web pages should contain the SAME HTML tags that you defined styles for in your style sheet. Once incorporated into each page, the pages should then look IDENTICAL in the way that they are laid out and displayed. However, the actual content must be different in each one. So, be sure to use the same HTML tags in each of the THREE pages. This means that there should be the SAME TAGS IN EACH PAGE and you should make sure that they are the same ones that you made you style assignments to in your style sheet!

In TWO of these pages, you will LINK your style sheet into them. These  will be the ONLY styles added to these two pages. DO NOT add any other styles to these two pages! In the third page, you will IMPORT in your style sheet and also add some EMBEDDED style code to change the property values for EACH of the tags’ styles that you created in your style sheet. Which styles are changed and how many are up to you! You just need to take the existing styles and make them different. DO NOT add any new properties here (only new values of existing style properties) and DO NOT add inline styles – only embedded!


So, upon completion of this lab, you should have a TOTAL of FOUR

files:  THREE web pages and  ONE style sheet. 

So, here are the basic steps to follow:

1.      Create a new folder in your Lab folder and name it, Styles (or whatever you would like to reflect the fact that it is for style sheets).


2.     Create your web page and lay it out using the 5 – 7 different tags that you have decided to use;

3.      Save it THREE times with a different file name each time (ie. Lab 6a.htm, Lab 6b.htm, Lab 6c.htm, etc.). You just need to follow the same naming convention that you have been using on your previous web page files.

4.     Go into the other two and change the content – NOT the layout. Each of the 3 web pages must have different content.

5.     LINK the style sheet to TWO of the pages.

6.    IMPORT the style sheet into the third.

7.     On the page with the IMPORTED styles, also add EMBEDDED styles to change one of the property values for each of the tags that you have styled (much like what you did in Lab 5. DO NOT USE INLINE STYL:ES… ONLY EMBEDDED FOR THE CHANGE!

8.     Upon completion, you will have three web pages and one style sheet. The web pages’ content must be different in ALL THREE PAGES, but the layout should be the same. Two of the pages will have exactly the same appearance and the third page will look similar but some slight modifications in styles.
