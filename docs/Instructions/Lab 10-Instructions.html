Lab # 10

 

In this lab, you will create a custom form that can be used as a questionnaire!

Your questionnaire should contain AT LEAST 10 QUESTIONS (but no more than 15).  The questionnaire can be about ANYTHING you wish; so your questions can be as “different” as you chose to make them. Feel free to be imaginative!

You MUST use each of the following FORM controls at least once in your questionnaire:  

·        Checkbox

·        Text

·        Textarea

·        Radio

·        Select  

 

In addition, with the inception of HTML5 several new input types have been added. You need to add the following new input types to your form too:

·        color

·        date

·        email

·        tel

·        url

 

In addition, the Questionnaire MUST contain both a SEND and a RESET button!

Please add the following attribute to your < form > tag:

              enctype = “text/plain”


Finally, you need to add some style to your FORM to “dress it up” so that it looks interesting and professional. You can add images, background color, borders, or whatever else you think will improve it’s look.  As always… be creative!
