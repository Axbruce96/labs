Adding external fonts to your web page:

 

a. Create a sixth paragraph and use a font imported from Google fonts to style your paragraph! Note: your paragraph should contain at least two sentences so that you can adequately display the new font.

 

b. Create a seventh paragraph and use a font that is being linked to via the “@font-face” rule. To make this one work, you need to carefully follow these rules:

·       Download a new, FREE font from the web and only download a font of the following type:

o   TTF

o   OTF

o   WOFF

·       Create a “ fonts ” folder and install the downloaded font into it;

·       Make certain that the font folder that you downloaded is UNZIPPED first. If you install your unzipped, downloaded font folder directly into your newly-created “fonts” folder you must remember that you have added another folder layer to your path and code it accordingly;

·       You must first define a name for the font ( ie. Lab7Font ), and then point to the font file in your style code; refer to the name of the font ( Lab7Font ) through the font-family property;

·       Use only lowercase letter in the url as not all browsers react the same when uppercase letters are used.

·       Example:   @font-face {  font-family: Lab7Font;
                                                  src: url(fonts/happy.woff); }

 

div { font-family: Lab7Font; }
