Lab # 4

 

Create a Web page that contains a Table Of Contents  at the top of the page. The Table Of Contents  will contain hypertext links to other areas of the SAME page. It can be named whatever you wish!

The page should contain at least FIVE (5) INTERNAL links that are set-up so that the listing in the Table  directs the User to it's corresponding link on the same page ; the following reflects the BASIC structure of your page set-up ( it does NOT need to look exactly like this – this is just an example ) :

I.      CHAPTER 1

II.    CHAPTER 2

III.   CHAPTER 3

IV.   CHAPTER 4
V.    BIBLIOGRAPHY

 

CHAPTER 1  Return To Table of Contents
                Some content here….


CHAPTER 2  Return To Table of Contents

                Some content here….

Etc., etc.


BIBLIOGRAPHY  Return To Table of Contents

 

Beside each "content section," create an additional link that will return the User back to the TABLE OF CONTENTS  at the top of the page.
